# User JSON Contracts

Put new user-authored compact JSON contracts in this directory.

The starter guide uses this location in examples such as:

```bash
cp new_user_umat_starter/templates/new_umat_minimal_template.json json_files/my_new_umat.json
```

Then run:

```bash
python3 new_user_umat_starter/scripts/check_config.py --config json_files/my_new_umat.json
```
